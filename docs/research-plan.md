# Nurolab Research Plan

## Vision

To develop an open, scalable, and research-oriented EEG analytics platform capable of supporting cognitive, neurological, and behavioral studies while advancing accessible neurotechnology.

---

# Phase 1: Software Foundation (Current)

## Objective

Build a complete EEG analytics platform using publicly available datasets.

### Goals

* EEG preprocessing
* Feature extraction
* Dashboard development
* Reporting system
* Machine learning experimentation

### Success Criteria

* End-to-end processing pipeline operational
* Dashboard deployed
* Reports generated
* Research documentation completed

---

# Phase 2: Hardware Integration

## Objective

Connect real EEG acquisition hardware to Nurolab.

### Proposed Hardware

* EEG Headband
* ADS1299
* ESP32

### Goals

* Real-time acquisition
* Wireless transmission
* Cloud ingestion
* Live visualization

### Success Criteria

* Continuous EEG streaming
* Stable cloud synchronization
* Real-time dashboard updates
